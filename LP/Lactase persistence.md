# 1. 核心定义（Phenotype）
- 乳糖酶持续性（LP）：成年后仍持续表达乳糖酶（lactase），能够消化乳糖
- 对照状态：乳糖酶非持续性（LNP）——断奶后乳糖酶表达下降（哺乳动物默认状态）

# 2. 生理机制（Mechanism）
- 乳糖（lactose）：乳汁中主要碳水
- 乳糖酶（lactase / LPH）作用：
  - 将乳糖 → 葡萄糖 + 半乳糖
- 如果缺乏乳糖酶：
  - 乳糖进入肠道发酵 → 腹胀、腹泻（乳糖不耐受）
