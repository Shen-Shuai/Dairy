# 1. 核心定义（Phenotype）
- 乳糖酶持续性（LP）：成年后仍持续表达乳糖酶（lactase），能够消化乳糖
- 对照状态：乳糖酶非持续性（LNP）——断奶后乳糖酶表达下降（哺乳动物默认状态）

# 2. 生理机制（Mechanism）
- 乳糖（lactose）：乳汁中主要碳水
- 乳糖酶（lactase / LPH）作用：
  - 将乳糖 → 葡萄糖 + 半乳糖
- 如果缺乏乳糖酶：
  - 乳糖进入肠道发酵 → 腹胀、腹泻（乳糖不耐受）

# 3. 全球分布（Pattern）
## 3.1 高频（>75%）
- 北欧 & 中欧（如芬兰、瑞典、英国等）
- 以及其移民后代（北美、澳洲）
## 3.2 局部高频（但不连续）
- 西非（如富拉尼）
- 东非（如马赛、图西）
- 中东（如贝都因）

# 4. 进化解释（Evolutionary Hypothesis）
## 4.1 文化—基因共进化
- 新石器时代（~10,000 BP）：
  - 动物驯化（牛、羊、骆驼）
  - 奶成为稳定资源
- 结果：
  - 能喝奶的人有生存/繁殖优势
  - LP频率上升
## 4.2 模型问题
- 有些牧民喝奶但LP很低（蒙古、中亚）
- 有些人不喝奶但LP很高（部分东非采集者）
- 欧洲 LP 中世纪前并不高
